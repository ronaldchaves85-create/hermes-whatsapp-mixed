Quero criar a skill `youtube-claude-code-lead-discovery`.

Objetivo:
Buscar leads no YouTube usando AIsa only, principalmente vídeos sobre Claude Code ou temas parecidos, priorizando canais menores e vídeos com tração.

A skill deve:

- aceitar uma palavra-chave do usuário
- buscar vídeos no YouTube via AIsa
- filtrar vídeos com views acima de um corte
- priorizar canais menores quando fizer sentido
- resolver o melhor match do canal antes de responder

Saída obrigatória para cada resultado:

- views
- título do vídeo
- inscritos
- thumb
- link do vídeo

Regras:

- usar somente AIsa
- não misturar browser ou outras fontes
- manter a saída pronta para Telegram
- se o canal vier ambíguo, verificar o melhor match antes de responder

A skill deve conter:

- descrição clara
- quando usar
- fluxo de execução
- formato de saída
- regras de query
- pitfalls
- checklist de verificação

No final, quero a skill validada e pronta para uso.
